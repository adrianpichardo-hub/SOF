# ISA - Generador de Statement of Facts

Herramienta offline para generar SOF a partir de logs de operaciones portuarias.

## Uso

1. Abre `index.html` en el navegador
2. Completa los datos de cabecera
3. Pega el log en el área de texto
4. Pulsa "Generar SOF"
5. Descarga el Excel

Funciona 100% en el navegador (sin backend).
